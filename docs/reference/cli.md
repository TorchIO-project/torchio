# Command-line interface

TorchIO includes a `torchio` command with several subcommands.

## Usage

```
torchio [plot|animate|info|convert|transform|cache] [options]
```

## Reference

The CLI is built with [tyro](https://brentyi.github.io/tyro/).
Help text for each subcommand is generated from the source code.
